# Ses Dosyaları

Bu klasöre aşağıdaki ses dosyalarını ekleyebilirsiniz:

- `correct.mp3` - Doğru cevap verildiğinde çalacak
- `wrong.mp3` - Yanlış cevap verildiğinde çalacak
- `thinking.mp3` - Soru okunurken arka planda çalacak
- `intro.mp3` - Oyun başlarken çalacak

Ses dosyalarını eklemek için bu klasöre kopyalayın.
