# 🎵 Ses Dosyaları

Bu klasöre şu 3 dosyayı eklemelisiniz:

1. **thinking.mp3** - Soru sırasında çalan müzik (40 saniye, loop)
2. **correct.mp3** - Doğru cevap alkış sesi (10 saniye)
3. **wrong.mp3** - Yanlış cevap dramatik sesi (3-5 saniye)

## Nasıl Bulunur?

YouTube'da "Kim Milyoner thinking music", "Kim Milyoner correct answer", "Kim Milyoner wrong answer" aratıp MP3 olarak indirebilirsiniz.

## Önemli

Dosya isimleri tam olarak şöyle olmalı:
- `thinking.mp3`
- `correct.mp3`
- `wrong.mp3`

Dosyaları bu klasöre kopyalayın ve oyunu yeniden başlatın.
